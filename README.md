
# 🚗 Car Price Prediction using Multiple Linear Regression

## 📌 Overview

This project implements a **Multiple Linear Regression model** to predict used car selling prices based on multiple real-world features.

It demonstrates a complete **end-to-end machine learning pipeline**, including data preprocessing, feature engineering, model training, and evaluation.

📊 **Final Model Performance:**  
R² Score: **0.8313**

---

## 🎯 Objective

To build a regression model that accurately predicts used car prices by learning relationships between multiple independent variables such as:

- Year of manufacture  
- Kilometers driven  
- Fuel type  
- Transmission type  
- Ownership history  
- Engine-related attributes  

---

## ⚙️ Workflow

### 1️⃣ Data Preprocessing
- Handled missing values  
- Removed irrelevant features  
- Cleaned inconsistent data  

### 2️⃣ Feature Engineering
- Converted categorical variables using One-Hot Encoding  
- Improved feature representation for model learning  

### 3️⃣ Model Building
- Implemented Multiple Linear Regression using Scikit-learn  
- Split dataset into training and testing sets  

### 4️⃣ Evaluation
- Evaluated model using R² Score  
- Analyzed prediction performance on unseen data  

---

## 📈 Results

| Metric | Score |
|--------|------|
| R² Score | 0.8313 |

✔ Model explains ~83% of variance in car prices  
✔ Strong baseline regression performance on real-world dataset  

---

## 🧠 Key Learnings

- Data preprocessing has more impact than model complexity  
- Real-world datasets require extensive cleaning  
- Encoding categorical variables is essential for regression models  
- Feature engineering significantly improves model accuracy
